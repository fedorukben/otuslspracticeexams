FAQ

Is this an official Ontario Tech resource?

Yes. Practice Exams is a study tool offered through the Teaching and Learning Centre's Student Learning Support team, developed to help students in math and statistics courses.

Where do the questions come from?

From a curated bank of original practice questions written to mirror the style and difficulty of typical course exams. Every attempt pulls a random selection.

Should I rely only on this tool to study?

No. This practice generator may not encompass all topics in your course. Use it as one part of your study plan, attend Exam Review workshops, and visit Math Study Hall for support when you are stuck.

What happens when I click Regenerate?

Regenerate builds a new random set of questions for the same course, exam, and question count. If you want to keep your current set, copy and save its OTEX code first.

What is an OTEX code, and when should I use it?

An OTEX code recreates one exact practice set. Use it to reopen your own set later or share the same set with classmates for group study.

My OTEX code is not loading. What should I do?

Make sure you copied the full code and did not remove characters. You can paste either the OTEX code itself or the full exam link that contains ?code=....

Why do I see Question IDs on each problem?

Question IDs are lookup keys that help instructors and support staff map each question to solution banks and references. Students can ignore them if not needed.

What if topics from class are not appearing here?

This bank may not cover every lecture topic. Keep using your course notes, assignments, textbook questions, Exam Review workshops, and Math Study Hall for full coverage.
